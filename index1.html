Programa en C++ que diferencie entre un dato de cadena de caracteres con un número entero.

Solicite al usuario que ingrese cadenas y números enteros.
Utilice la función isdigit() de la biblioteca para verificar si el valor ingresado por el usuario es un número entero. Si el valor no es un número entero, muestre un mensaje de error y brinde al usuario dos oportunidades más para ingresar un número entero válido.
Si el valor ingresado es un número entero, será la diferencia entre la longitud del número entero y la cadena. Mostrar los resultados diff al usuario.
Este código usa un bucle do-while para permitir que el usuario ingrese un número entero válido hasta tres veces. La función cin.fail() verifica si la entrada es un número entero y la función cin.peek() verifica si hay caracteres adicionales en el búfer de entrada. Si el valor ingresado no es un número entero válido, el búfer de entrada se borra y se muestra un mensaje de error al usuario. Si el usuario no ingresa un número entero válido después de tres intentos, el programa muestra un mensaje de error y sale con un código de retorno de 1. De lo contrario, el programa calcula la diferencia entre números enteros y grados. Resultados a los usuarios.
